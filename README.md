# 曾仲仙个人求职网站使用说明

## 项目概述
基于黑白灰高级风设计的个人求职网站，采用左右布局，突出数据运营专员求职竞争力。所有交互流畅、轻量化，适配PC/平板/移动端。

## 文件结构
```
Xe/
├── index.html          # 主页面文件
├── styles.css          # 样式文件
├── script.js           # 交互脚本
├── README.md           # 使用说明
└── images/             # 图片资源文件夹
```

## 功能特性

### 1. 核心布局
- **左侧固定栏（280px）**：个人头像、基本信息、导航菜单
- **右侧内容区**：5个主要板块，支持无缝切换
- **响应式适配**：平板（768-1199px）和移动端（≤767px）

### 1.1 用户体验优化
- **板块切换**：平滑滚动+0.3s过渡动画
- **回到顶部**：移动端右下角圆形按钮，滚动后显示
- **懒加载**：所有图片支持懒加载，优化加载速度
- **页脚信息**：底部固定显示求职信息和更新时间

### 2. 主要板块
- **个人简介**：核心信息、优势展示、联系方式，新增技能标签和简历下载
- **实习经历**：3份大厂实习经历，带量化成果和官方Logo
- **个人技能**：专业技能和通用技能分类展示
- **作品展示**：3个代表性项目，支持详情查看
- **证书荣誉**：重要证书和奖项列表

### 2.1 求职转化优化
- **简历下载**：个人简介板块添加PDF下载按钮
- **技能标签**：新增"Python可视化""跨境数据运营""RFM分层建模""英文办公"标签

### 3. 交互功能
- **板块切换**：无刷新流畅切换，带过渡动画
- **二维码放大**：点击微信二维码可放大查看
- **一键联系**：邮箱支持`mailto:`跳转，电话支持`tel:`拨打
- **作品详情**：点击"查看详情"显示完整项目信息
- **键盘快捷键**：数字键1-5快速切换板块，ESC关闭弹窗

## 样式自定义

### 1. 颜色配置
**文件**：`styles.css`
**主要颜色变量**：
- 主背景：`#F5F5F5`
- 左侧栏背景：`#2C2C2C`
- 核心文字：`#000000`
- 正文文字：`#333333`
- 辅助文字：`#666666`

### 2. 响应式断点
**文件**：`styles.css`
**媒体查询位置**：
- 平板：`@media (max-width: 1199px)`
- 移动端：`@media (max-width: 767px)`

### 3. 动画效果
**文件**：`styles.css`
**主要动画**：
- 板块切换：`fadeIn 0.3s ease-in-out`
- 卡片悬停：`box-shadow 0.3s ease, transform 0.3s ease`
- 按钮悬停：`background-color 0.3s ease`

## 性能优化建议

### 1. 图片优化
- 压缩所有图片文件，建议单张不超过200KB
- 头像建议使用正方形裁剪
- 二维码建议尺寸：80px×80px

### 2. 加载优化
- 图片已实现懒加载
- 使用CDN托管静态资源（如需要）
- 启用Gzip压缩（服务器配置）

## 部署说明

### 1. 本地预览
直接双击`index.html`文件即可在浏览器中预览。

### 2. 网络部署
将整个文件夹上传到任意Web服务器即可，无需额外配置。

### 3. GitHub Pages部署
1. 将代码上传到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择主分支作为源
